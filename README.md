# Countdown Program

## Overview

The Countdown Program is a web-based application built using HTML, CSS, and JavaScript. It continuously counts down each calendar year until the year ends. Once a year is completed, the program automatically resets and starts counting down the next year.

The countdown is designed to run indefinitely, transitioning from one year to the next without requiring any manual updates. According to the project's concept, the program will continue operating until the end of the world or the return of Christ Jesus.

## Features

* Real-time countdown showing:

  * Days
  * Hours
  * Minutes
  * Seconds
* Automatically detects the current year.
* Automatically switches to the next year's countdown when the current year ends.
* Infinite looping from year to year.
* Responsive design for desktop and mobile devices.
* Clean and simple user interface.

## How It Works

1. The program detects the current date and time.
2. It calculates the remaining time until the end of the current year.
3. The countdown updates every second.
4. When the countdown reaches zero, the application automatically begins counting down the next year.
5. This process repeats continuously without any manual intervention.

## Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/countdown-program.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.

No additional dependencies or installations are required.
